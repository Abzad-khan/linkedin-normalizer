# Voyager Profile Normalizer Candidate Pack

This folder is safe to share with participants.

## What is included

- 3 public input fixtures
- the expected output contract
- a quick pass checklist
- 1 sample normalized output for a public fixture

## What is intentionally not included

- hidden fixtures
- all expected outputs
- scoring scripts
- internal evaluator logic

## Files

- `fixtures/profile_1.json`
- `fixtures/profile_2.json`
- `fixtures/profile_3.json`
- `OUTPUT_CONTRACT.md`
- `SUBMISSION_CHECKLIST.md`
- `sample_output_profile_1.json`

## How to use this pack

1. Read `OUTPUT_CONTRACT.md`
2. Test your script against the 3 public fixtures
3. Compare your result format with `sample_output_profile_1.json`
4. Review `SUBMISSION_CHECKLIST.md` before submitting

## Share Note

These public fixtures are representative examples only. Final evaluation may include additional hidden cases that follow the same normalization rules and output contract.
